24考研双非一战22408失败贴&回忆贴

24考研双非一战22408失败贴&回忆贴

前言

本人坐标山东某双非，绩点3.2/5，四级532，六级475，高数线代期末60+，数据结构73，计组88，操作系统93，计网92。一战22408中海洋，67+71+87+92，总分317，以初试倒数第一复试倒数第二最终被刷。复习时长2022年12月28日 - 2023年12月23日，寒假只学了高数和英语，开学后3月份开始408,5月份开始用番茄todo计时，到考前共1500小时。自认为在过去一年的备考中走过很多坑，故写下此贴供大家参考，因自认为有概率二战，所以在写规划时也会偏向二战的时间结点一些。

声明

在考完一次后，我认为找到自己适合的学习方法和进度规划太重要了，可以参考别人的经验贴，但一定要根据自己情况具体问题具体分析。在下文的分享中，我说的很多规划都是以“如果我再考一次，我会走哪些路，避开哪些坑”，而这些方法不一定适合于你，所以在上文我贴出了我的背景以及复习情况，供大家参考。

政治（67）

我的复习路线大概是，8月底开始听徐涛强化班，一直磨蹭到10月初才把马原听完，10月转战腿姐技巧班，只听了选择，模拟卷出来之后一直刷选择，一天1-3套，最后肖四出来直接背研木易的精缩和张修齐的当代。在听完徐涛马原之后，我的进度安排完全按照b站up”自动化土狗“的规划来走的。总结一下我的复习规划就是徐涛马原 + 腿技巧班 + 模拟卷选择 + 肖四精缩，实际上在复习过程中，我觉得前期看马原的作用也微乎其微，因为腿姐的技巧班会给你再稍微串一下基础知识，因此我认为一个考数学 + 408的政治复习思路可以是10月份直接上腿姐技巧班，在模拟卷出来之前做做腿姐讲义上的题就可以，模拟卷出来之后疯狂刷选择，可以用小程序辅助，自己笔记整理一下各种帽子题，等肖四出来直接背研木易的精缩，今年肖老几乎全压中了，最后一题当代背张修齐的模版，考场上默写就完事。这样一个规划完全可以以一个较少的工作量，把政治考到一个不拉分的分数，市面上目前推荐的核心考案 + 1000题，个人认为没有买的必要。

英语（71）

考这个分数的确是发挥失常了，可能是板凳质量的问题，当时考场上整个教室的板凳吱吱响，有人稍微挪动一下身子板凳就会响，自己就很不幸被影响到了。写到这了就顺便再写一下考场状态吧，拿到试卷先扫了一下作文，小作文是个啥信忘记了，反正不是通知，大作文写一个学校劳动课主题的，当时感觉和自己背的模版不太匹配，自己发挥了点加上模版把作文写完后开始做阅读，就是这个时候开始注意到教室的声音，第一篇阅读做了很久都没做出来一题，当时非常焦虑，就翻了一页开始做第二篇，第二篇还是一样的情况，被影响的真的非常严重了，然后自己用卫生纸揉了两团耳塞堵住耳朵，直接跳到了新题型开始做，所幸新题型没被卡住，而且做完之后自己的状态也恢复了一些，之后依次做完了翻译，阅读4,3,2,1，完型，没错，阅读我直接倒着做了。考完感觉直接寄了，英语从没做的这么差过，自己考前英语一直按80+计划的，最后出分71，比我想象的要好一些，由此可见，真正拉分的还是数学和专业课呀。

再说一下自己的复习思路吧，英语我算是开始的非常早的，大三上备考六级的时候，我就开始背红宝书，当时用的”不背单词“背的，大概寒假之后我就把第一轮单词过完了，寒假看了田静老师的句句真研，个人认为英语基础可以且考英二的，语法课没必要听。之后就是背单词，3-7月我先是用纸质版《闪过》把高频词背了一遍，之后就用”墨墨背单词“开始背第二轮红宝书，这个过程持续到考前，暑假开始刷阅读，先是做了04-10年的英一，之后听了唐迟的阅读课，9月份开学后开始做10年后的英二，英二每张大概错1-5个，个人认为英二在17年18年之后的难度是要小于之前的，阅读做到最后的时候我基本错1-3个，当时还是挺有自信的，但24年的难度个人感觉有所提升。在10月份左右刷完阅读开始做准备翻译、完型和新题型，这三个我准备的都非常少，个人觉得翻译还是应该准备一下，其他两个阅读做好了自然也会了。11月中开始准备作文，作文直接背的晶婷老师的作文书，加上自己在b站搜了点句子凑了一个模板，下面是我认为自己在英语这个科目踩得一些坑：

  1. 单词方面：用墨墨背完二轮红宝书后，每天的单词量只有墨墨推的复习，但按墨墨的记忆算法，我背到后期有些单词的下次复习时间是100天后，200天后，在最后冲刺阶段，我感觉我的单词量是一直在下降的。
  2. 阅读方面：后期的做题量不足，个人觉得即使在冲刺阶段，两天要拿出一天来做两篇阅读，以此来保持手感。
  3. 翻译方面：时间分配不足，自己只听了晶婷老师的一个40分钟的翻译课，翻译五六个真题就上考场了。
  4. 作文方面：时间分配不足，小作文模板在考前几天才赶出来，自己写的文章太少；大作文中间段背的例句太少，以至于考场需要自己发挥。
  5. 压力方面：平时做阅读时没有将自己置于考场环境下，后期自己只掐点完整做了两套真题，且没有严格按照考场规范，以至于在考试时乱了阵脚

总结以上，我认为一个英二复习的方法可以是：

  1. 前期一直背单词，每天30min - 1h30min不等
  2. 中期开始阅读，可以参考唐迟的阅读课，每天任务量不必太多，两篇即可，目前英二10-24共60篇阅读，最快可以一个月刷完
  3. 新题型、完型：做完阅读可以直接做，10-24共30道题
  4. 翻译：个人认为做题 > 听课，真题起码要做一遍，上考场前保证手感存在，这个阶段在做翻译的同时可以二刷阅读
  5. 作文：感觉11月就可以准备了，不然后期肖四 + 作文有点吃不消，作文非常重要，后期一定要自己动笔写，在考场上一般先把作文写了，顺利写完作文做剩余的题目也轻松一些。
数学（87）

先说我的复习历程：

  • 寒假 - 3月底：高数基础，武忠祥基础班，题目张宇基础300题

  • 4-5月：线代基础，张宇基础30讲线代部分，题目张宇300题，之后做了些660的线代

    • 张宇线代基础神中神
    • 660线代太简单，不建议做
    • 在这个阶段我做了一部分660高数的一阶题目，但是效率极低，一晚上只能做10个题且正确率不高，遂及时止损，准备高数强化

  • 5.11-8.23：高数强化，武忠祥强化班，习题严选题+660

  • 7 - 10：同步线代强化，张宇9讲，习题1000题AB组，之后刷了880的特征值和二次型

  • 9-10月中：880高数部分，配套咋子哥讲解

  • 10月中-11月中：09-23真题，同步二刷880错题，但是没刷完

  • 11月中-考前：模拟题，李6李4，张八两套，同步看kira的线代醒脑课（没听完）和咋子哥的线代救命课，完成部分真题错题二刷

    • 咋子哥李4李6神中神，咋哥真是冲刺阶段的光了
    • kira神中神，不仅是醒脑课，1000题讲的也很好，老师也很懂梗，听课很愉快
    • 咋子哥二次型醒脑课很强，不过最后没考这么难的
    • 线代24真题应该是满分，这两个老师真的很好

  • 整个规划大多参考了知乎“邂逅遗憾”的经验贴，给邂逅哥丢脸了😭

数学这门科目是我觉得踩坑最多的一门，在学之前我看了很多经验贴，自己做的题也很多，张宇300、660高数、880高数、严选题、1000题线代AB组我都做了，660线代和880线代做了部分，其实这样恰恰是问题所在，下面总结一下我认为的不妥之处：

  1. 习题不用刷太多，感觉两本就够了，线代感觉做一本就够了，但是都要二刷三刷，880高数和线代质量都不错。自己在刷题时一昧求进度，仿佛目的不是学会知识，而是刷完这本习题册，况且自己每一本习题的正确率都很低，盲目的做一本右移本真是太蠢了；
  2. 没必要死磕证明题，880在函数和积分章节都有大片的证明题，当时一刷880时第三章的证明题我好像做了3天才做完，后来看真的完全没有必要，等你发现真题中一些基本题你都搞不定，就会无比后悔当初花这么久去做证明题；
  3. 模拟题：做的太少了，11月中开始刷模拟卷感觉有些晚，最好11月初甚至10月底就开始，今年李4李6和真题难度不匹配，当时没有意识到，都推荐李林就去做了，做了张八第一套分数很低，当时网上都说张宇卷子偏难，自己也没在意，自认为真题不会这么难；
    • 所以模拟卷真的需要上强度啊！多做几个卷子，多做几个老师的，不要听说题难就不去尝试，停在李林的舒适区上考场直接降智
    • 做卷子时严格规划草稿纸使用，严格掐点，尽量不中断
  4. 做题心态：这个太重要辣，triple star。在平时做题时，习题也好套卷也好，特别是小题，你算出一个结果一定是要经过一个非常仔细的计算得出，而不是“应该是这个结果”这样一个心态。特别是在做习题时，有时为了方便，自己有思路后就在草稿随便写了写，得出了一个大概的“答案”，对答案后发现正确就把这题过去了，我去年在做题时经常这么干，然而在考场的高压环境下，你平时的做题习惯会直接被带到考场，唯有平时练习注意，才能在考场上避免悲剧发生。今年真题第一题就被我这么模棱两可算过去了，加上5分可能最后的结果也不一样了罢。

结合以上，我得出了这样一个复习思路（主要针对于二战规划）

  1. 前期看强化课，同步做880/1000，后续这本要二刷三刷，证明题部分可以做第一问，其他直接放
  2. 开套卷前可以一直刷题，有时间的话可以刷第二本习题
  3. 套卷尽早开，最好11月前可以完成真题，开始模拟卷
  4. 模拟卷需要上强度，往年张宇、余炳森、超越卷的评价都偏难，需要尝试，不可局限于李林
  5. 推荐课程
    • 张宇线代基础：很强，会给你讲一些本质的东西，我听过李永乐老师的线代基础，感觉远不如张宇基础
    • 没咋了880
    • kira醒脑，不知道能不能直接按强化课去听，待研究
    • 没咋了线代救命：若最后线代有漏洞且时间不多可以听
408（92）

复习历程：

  • 3月-4.15：数据结构一轮

  • -5.22：计组一轮

  • -6.21：操作系统一轮

  • -7.20：计网一轮，至此408一轮结束，主要方式是王道视频+课后选择题

  • 7.20 - 10月20：强化，看王道强化课，做大题，期间做了4个思维导图

  • 之后就是刷套卷，同步强化未完成的内容，一边刷卷子一边补不会的，平均3天一套卷子

直接总结坑点吧：

  1. 自强化开始后节奏直接乱掉
    • 数据结构代码题在强化时未复习，后期一天用20分钟左右做一题，但因去年和前年都考了图，自己存在侥幸心理未复习图，遂寄，代码题尽量强化前期就开始做，避免占用后期过多时间
    • OS和计网未强化完就开套卷了，当时存在进度焦虑，10月底强化未完成便开始套卷，之后的时间强化和套卷穿插，效率很低
    • 以上两点导致做卷子时，算法题不会直接空，文件大题、PV大题不会的直接空，再加上有些大题听后还忘了，导致套卷做的非常惨
  2. 大题题量不够：真题的大题大多做了1-2遍，且没有做王道书的其余大题
  3. 小题二刷未完成，理想情况下尽量二刷，真题小题可以三刷

再说一下我认为备考过程中比较有用的方法：

  1. 思维导图：我是自己做了一份，后期冲刺基本上全靠思维导图回忆
  2. 湖科大计网模拟：有408命题风格的感觉，只是没想到今年的计网比湖科大还偏
  3. 每一科单独笔记：可以写一下一直忘记的重难点，408全靠反复多遍记忆

基于此，我给出我认为一个可以的408复习思路

  1. 反复多遍，基础至少过3遍，第一遍看课，后面就可以直接看导图了；
  2. 二轮大题跟紧王道，强化完成再开套卷，算法题需要这个阶段准备个大概，后期可以进行补充；
  3. 注意一下进度，进度落后则多分配一些时间，尽量10月中旬前完成强化开始真题；
  4. 做卷子时查缺补漏，把忘记多次的知识点或是值得注意的点写到笔记上；
  5. 尽量留时间做一些模拟题，听说今年竟成和零壹的模拟卷质量都不错。
环境篇
  • 住宿：个人认为备考环境是非常重要的，我有一点轻度的神经衰弱，恰好宿舍又有一位晚上打游戏白天睡觉的老哥，虽然宿舍可以做到12点熄灯闭麦，但是只凭笔记本风扇转动的声音和鼠标的声音就让我一直睡不着，10月11月最严重时一周就会有一两天一夜睡不着。之后通过沟通，舍友换了静音鼠标，加了隔音窗帘，但是风扇问题还是一直解决不了，没办法就忍到了考研结束，个人认为睡眠不好是我一战失利的一个非常重要的因素。在备考之初我尝试过去租房住，但最后因为各种因素没有租成，加上父母劝我要改变自己心态，就想着在宿舍呆这了，然而事实证明心态真的很难改变，而且我认为一个嘈杂的环境和自己的神经衰弱是一个恶性循环，环境越差神经衰弱越会加重，很难做到自我调节。
  • 酒店：还有一些涉及到环境问题的就是初试前订酒店，尽量订个好一些的吧，当时为了省一两百块钱订的酒店空调噪音很大，第一晚一两点才睡着，加上第二天中午来不及睡觉，所以考英语时整个人已经很疲惫了。
心路历程

最后谈谈自己的心路历程吧，也算是对大学四年做一个总结。初来大学时跟着舍友一直玩，大一打了一年的游戏，空闲之余参加了学院的辩论社，但也没拿到什么成绩，大一期末挂了C语言和离散数学；大二开始依旧是玩，到期末前一个月觉得不能再挂科了，开始突击四级和期末，最后都考得不错；大二下开始做规划，当时对计算机专业的就业一无所知，在网络上浏览计算机专业的就业方向，看着视频里说的前端 后端 客户端一头雾水，直到在b站无意间刷到一个C++开发的学习路线，我看到博主提到的很多要学的科目都是大学的必修课，且该博主的学习路线是从大一就开始的，从大一到大四要干什么直到拿到offer，所以当时的我以为从现在开始准备校招已经为时已晚，加上自己想要提高一下学校层次，接受更高等的教育，就把视线放到了考研这条路。

因为当时也不知道自命题的存在，以为所有学校都是考408，我深知自己基础很差，于是在大二下学期我跟着学校好好学了一遍计组，数据库，自己重新学了一下C语言，顺便又备考六级，但是这次没过。大三上学校开设计网和操作系统，我提前买了一本top-down计网在假期读，开学后自己学数据结构，计网把中科大郑烇老师的课看完了，操作系统也过了一遍王道的课，同时第一次用C++做了一个学校的管理系统课设，这一学期我开始写博客，每天学完就将所学内容记录到博客上，最后也备考了一下六级，侥幸过了。这一学期大概是我大学里学的最开心的一个学期，没有什么其他事情干扰我，也没有进度要求，每天的学习量根据自己心情而定，同时这学期我第一次了解到科软的存在，无导师+实习+华五title非常吸引我，于是暗暗下定决心要考这个学校。

之后就是长达一年的备考，我本以为自己提前把408好好学了一遍，在备考时会轻松些，但事实是基本全部忘干净了，在第一轮复习感觉和零基础没什么区别。其次就是我的高数，因为基础太差在前期看课的过程也是相当痛苦，第一遍看武老师时有很多东西听不懂，然后就去b站找对应的知识点去补，就这样磕磕绊绊过完了基础。在报名前后，我的数学真题分数平均在90-110之间，408刚刚做了几套真题也是90-100之间，觉得报考科软已经是痴人说梦了，遂跳车中海洋。

考后当天晚上在酒店就估了下分数，当时保守估计只有300分，参考去年的复试线330，且今年出来后大家都说题难，所以当时认为还是有概率能够进入复试。回家后本想一边准备复试一边准备工作，却发现自己哪个都抓不好，于是直接学C++准备工作。2.26号查了成绩317，排名24，按去年招10进11，今年招14，我判定基本无缘复试，难过了两天后回校准备春招，最后投了半个月左右拿了济南的一个9k*13的offer，几乎没有空挡接着就去实习。实习一周左右，在3月底海大出复试线，发现自己竟然压线进了，但当时的我白天需要在上班，而且只有一周就要复试，并且这周我和另一个实习生需要接受公司培训，为了保住工作我没有选择请假，只能用一个周末和每天晚上的时间复习，最后理所当然的寄了。

在找到工作后，公司的氛围很不错，同事对我也很好，本以为自己会放下考研，在济南打拼几年攒些钱，但去了一趟复试后还是被好学校深深的吸引到，一想到自己备考一年多最后没有结果也久久不能释怀，所以最近又萌生了二战的念头。当下计划还是工作到毕业吧，攒点生活费，在工作之余去健身房减减肥，考研一年胖了有20斤，等毕业之后再做其他打算。

以上就是本贴的全部内容，感谢你能看到这里。考研的确是一件很难的事情，祝你，祝我都能在考研这条路上收获自己所想要的，最终拥有一个锦绣的前程。

又想到一些想说的，写在这里

  1. 关于背单词时间：结合自身体会加上浏览了一些贴子后，我觉得把战线拉长是个不错的选择，一天半小时的工作量，复习+新学应该可以达到100~200。在我4月份左右用墨墨背红宝书时，一天要新学一个list加上复习墨墨推送的，每天背单词的时间可以达到一个半小时，不仅在前期占用了过多时间，到后期没有新单词背了自己也没清空记录，一直复习推送导致后期单词量一直下降，所以现在想想把战线拉长应该是个不错的选择。不过你如果单词基础薄弱，那还是建议前期多背一些方便开阅读吧。
  2. 对于完美主义看法：第一次听说这个词应该是在10月下旬未开套卷的时候，Kira老师说“没有必须看完的课程，也没有必须做完的习题”，在思考了一番后，408强化未完成我就开了真题，后期强化和真题并行运作。这么做感觉还是有些用的，主要是可以减缓你的进度焦虑，但我觉得更优解是在前期和中期保证进度正常，能够有时间完成所有事情然后再去开真题，至少对于我来说，拒绝完美主义只是对我强化进度落后的一种弥补。
